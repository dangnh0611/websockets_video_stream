# websockets_video_stream
python3 websockets video stream sample

## Virtual environment and dependencies
```bash
python3 -m venv venv
source venv/bin/activate 
pip3 install -r requirements.txt
```

## Start websocket server
```bash
$ python3 server.py
```
## Start websocket client
```bash
$ python3 client.py
```
